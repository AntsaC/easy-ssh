# Easy ssh
This is a simple and reusable ssh client cli based on ``yargs`` and ``ssh2``
## Requirements
- Node --v.10.16.0 or newer
## Installation
```
npm i -g easy-ssh
```
## Simple usage
### Create new connection 
```
easy-ssh new <name>
```
### Connect to ssh 
```
easy-ssh connect <name>
```
### List all connection 
```
easy-ssh list
```
### Help
```
easy-ssh --help
```
